	2. DFS Replication
Install those role eatch server which select for DFS Services

DFS Configuration:

	2. Crate a namespace. After create and namespace server (in ever server namespace is visible)
	3. Add all share Folder.

DFS Replication:
	1. Crate a replication group (grup member: all dfs server; primary server which belongs share folder)
	1.1. If share folder belongs difference server (than now add new replicated folder and selecet primary server which belongs folder and complete process)
